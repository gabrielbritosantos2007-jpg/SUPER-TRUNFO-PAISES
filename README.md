# SUPER-TRUNFO-PAISES
Este projeto foi desenvolvido para a faculdade utilizando a linguagem C, com o objetivo de aplicar conceitos fundamentais de programação como variáveis, estruturas condicionais (if/else), entrada e saída de dados (scanf e printf) e lógica de comparação.

O programa simula um jogo de Super Trunfo com cartas de países, onde cada carta possui informações como país, estado, cidade, população, área territorial, PIB e quantidade de pontos turísticos. O usuário escolhe uma carta para visualizar seus dados e o sistema realiza comparações entre as cartas para identificar qual possui os maiores atributos.

Funcionalidades:

Exibição de cartas de países
Comparação de população
Comparação de área
Comparação de PIB
Comparação de pontos turísticos
Exibição dos resultados em formato organizado

Tecnologias utilizadas:

Linguagem C
Biblioteca padrão stdio.h

Esse projeto foi desenvolvido com foco no aprendizado e prática da lógica de programação e estruturação de código.
